# Stege Status — data

Live data for the Stege Status TV board (opening hours + Stege markets).
`data.json` is regenerated weekly by a scheduled Claude agent (re-research)
and pushed here. The TV app fetches the raw URL over normal wifi:

https://raw.githubusercontent.com/beatsbyjaydk-tech/stege-status-data/main/data.json

Schema: see the `meta.note` field inside data.json.
